## **Název projektu**
**Přehled všech známých herních enginů a výhody či nevýhody každého enginu**

## **Žák**
**Tomáš Maršál**

## **Stručný popis tématu**
Webový přehled většiny všech známých herních enginů. Stránka bude one-page web se sekcemi, které ukážou popis, výhody nebo nevýhody každého enginu, tak aby si každý při vybírání herního enginu vybral ten nejvhodnější pro jeho projekt. Obsah bude načítán z JSON. 

## **Cílová skupina**
Vývojáři hledající nejvhodnější herní engine pro práci na jejich projektu.

## **Návrh obsahových sekcí**
- **Úvodní sekce** – krátké vysvětlení, co je herní engine a proč je důležitý.  
- **Seznam enginů** – jednotlivé bloky pro každý engine (např. Unity, Unreal Engine, Godot, CryEngine, GameMaker Studio, Construct, Panda3D, atd.).  
  - Popis enginu  
  - Výhody  
  - Nevýhody  
- **Porovnávací tabulka** – přehledná tabulka s hlavními parametry (jazyk, cena, platformy, komunita).  
- **Doporučení podle typu projektu** – např. pro mobilní hry, 3D AAA hry, indie projekty.  
- **Závěr** – shrnutí a doporučení, jak si vybrat správný engine.  

## **Návrh designu / wireframe**
- **One-page layout** s horizontální navigací (scroll nebo sticky menu).  
- **Hero sekce** s názvem projektu a krátkým popisem.  
- **Karty enginů** – každá karta bude mít logo enginu, krátký popis, seznam výhod a nevýhod.  
- **Porovnávací tabulka** – responzivní tabulka s možností filtrování.  
- **Barevná paleta** – moderní, minimalistická (např. tmavé pozadí + kontrastní barvy pro zvýraznění).  
- **Wireframe struktura:**
  1. Header (název projektu, menu)  
  2. Úvodní text  
  3. Sekce s kartami enginů  
  4. Porovnávací tabulka  
  5. Doporučení  
  6. Footer (autor, zdroje)  

## **Použité technologie**
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).  
- **Datová vrstva:** JSON soubor s informacemi o enginech.  
- **Frameworky/knihovny:**  
  - Možné využití **React** nebo **Vue.js** pro dynamické načítání obsahu.  
  - **Bootstrap/TailwindCSS** pro rychlý responzivní design.  
- **Verzování:** Git + GitHub.  
- **Deployment:** GitHub Pages nebo Netlify.  

## **Očekávaný přínos / co se naučím**
- Naučím se pracovat s **JSON daty** a dynamicky je zobrazovat na webu.  
- Procvičím si **frontendový vývoj** (HTML, CSS, JS).  
- Získám zkušenosti s **designem a strukturováním obsahu** pro přehledné webové aplikace.  
- Naučím se porovnávat různé technologie a prezentovat je uživatelsky přívětivým způsobem.  
- Vyzkouším si **nasazení projektu online** (GitHub Pages/Netlify).
